Beethoven - YouTube to MP3 Downloader
Version: 1.0

Developed by Dotsilicon Limited

----------------------------------------
DESCRIPTION
----------------------------------------
Beethoven is a desktop application that allows users to download and convert YouTube videos into MP3 audio files with multiple quality options.

The application automatically detects copied YouTube links from the clipboard and processes them into a download queue.

----------------------------------------
FEATURES
----------------------------------------
- Automatic clipboard detection for YouTube links
- Support for single videos and playlists
- Playlist downloads are saved in separate folders
- Multiple audio quality options (96kbps to 320kbps)
- Queue-based download system
- Pause / Resume / Stop controls
- Subscription-based access control
- Splash screen on startup
- Clean Solar-based UI

----------------------------------------
REQUIREMENTS
----------------------------------------
- Windows OS
- Internet connection
- FFmpeg (must be installed and available in system PATH)

----------------------------------------
HOW TO USE
----------------------------------------
1. Run Beethoven.exe
2. Complete subscription (first time only)
3. Copy any YouTube video or playlist URL
4. The app will automatically detect and queue downloads
5. Files will be saved in the selected download folder

----------------------------------------
NOTES
----------------------------------------
- Do not remove the "Resources" folder
- Do not rename internal folders
- Ensure FFmpeg is properly installed
- Internet is required for subscription validation
